# 課名：鴻門宴
# 朝代：漢魏六朝
# 作者：司馬遷

## 課文原文
（節錄）
沛公軍霸上，未得與項羽相見。沛公左司馬曹無傷使人言於項羽曰：「沛公欲王關中，使子嬰為相，珍寶盡有之。」項羽大怒... (中略) ...項伯乃夜馳之沛公軍，私見張良，具告以事... (中略)
沛公旦日從百餘騎來見項王，至鴻門，謝曰：「臣與將軍戮力而攻秦，將軍戰河北，臣戰河南... 今者有小人之言，令將軍與臣有郤。」項王曰：「此沛公左司馬曹無傷言之；不然，籍何以至此？」項王即日因留沛公與飲。項王、項伯東向坐；亞父南向坐—亞父者，范增也；沛公北向坐；張良西向侍。范增數目項王，舉所佩玉玦以示之者三，項王默然不應。范增起，出召項莊，謂曰：「君王為人不忍。若入前為壽，壽畢，請以劍舞，因擊沛公於坐，殺之。不者，若屬皆且為所虜！」莊則入為壽。壽畢，曰：「君王與沛公飲，軍中無以為樂，請以劍舞。」項王曰：「諾。」項莊拔劍起舞，項伯亦拔劍起舞，常以身翼蔽沛公，莊不得擊。
於是張良至軍門見樊噲。樊噲曰：「今日之事何如？」良曰：「甚急！今者項莊拔劍舞，其意常在沛公也。」噲曰：「此迫矣！臣請入，與之同命。」... (中略: 樊噲闖帳) ...項王曰：「壯士！賜之卮酒。」...「壯士！能復飲乎？」樊噲曰：「臣死且不避，卮酒安足辭！...」項王未有以應，曰：「坐。」樊噲從良坐。
坐須臾，沛公起如廁，因招樊噲出... (中略: 沛公逃脫)
沛公已去，間至軍中。張良入謝，曰：「...沛公不勝桍杓，不能辭。謹使臣良奉白璧一雙，再拜獻大王足下；玉斗一雙，再拜奉大將軍足下。」項王曰：「沛公安在？」良曰：「聞大王有意督過之，脫身獨去，已至軍矣。」項王則受璧，置之坐上。亞父受玉斗，置之地，拔劍撞而破之，曰：「唉！豎子不足與謀！奪項王天下者，必沛公也。吾屬今為之虜矣！」

## 試題範例
1. (單選) 在〈鴻門宴〉座次中，項羽「東向坐」，劉邦「北向坐」，這反映了當時何種形勢？
   (A) 項羽尊而劉邦卑
   (B) 賓主盡歡，不分尊卑
   (C) 劉邦為主，項羽為客
   (D) 雙方平起平坐
   解析：(A)。古代室內座次以東向（坐西朝東）為最尊，其次南向，再次北向。項羽東向，劉邦北向，顯示項羽自居尊位，劉邦處於臣服地位。

2. (非選) 樊噲闖帳後的發言，如何成功化解了項羽的殺意？
   參考答案：
   樊噲指責秦王暴虐，並強調劉邦「還軍霸上」是為了等待項羽，有功無過，卻聽信小人讒言欲殺有功之人（是「亡秦之續」）。這番話義正詞嚴，抓住了項羽「自矜功伐」且不願揹負「不義」名聲的心理。
