# 課名：桃花源記
# 朝代：漢魏六朝
# 作者：陶淵明

## 課文原文
晉太元中，武陵人捕魚為業。緣溪行，忘路之遠近。忽逢桃花林，夾岸數百步，中無雜樹，芳草鮮美，落英繽紛。漁人甚異之。復前行，欲窮其林。
林盡水源，便得一山，山有小口，彷彿若有光。便捨船，從口入。初極狹，才通人。復行數十步，豁然開朗。土地平曠，屋舍儼然，有良田、美池、桑、竹之屬。阡陌交通，雞犬相聞。其中往來種作，男女衣著，悉如外人。黃髮垂髫，並怡然自樂。
見漁人，乃大驚，問所從來。具答之。便要還家，設酒殺雞作食。村中聞有此人，咸來問訊。自云先世避秦時亂，率妻子邑人來此絕境，不復出焉，遂與外人間隔。問今是何世，乃不知有漢，無論魏晉。此人一一為具言所聞，皆嘆惋。餘人各復延至其家，皆出酒食。停數日，辭去。此中人語云：「不足為外人道也。」
既出，得其船，便扶向路，處處誌之。及郡下，詣太守，說如此。太守即遣人隨其往，尋向所誌，遂迷，不復得路。
南陽劉子驥，高尚士也，聞之，欣然規往。未果，尋病終。後遂無問津者。

## 試題範例
1. (單選) 關於「桃花源」社會的描述，下列何者正確？
   (A) 是一個長生不老的神仙世界
   (B) 是一個沒有戰亂、自給自足、人人平等的理想社會
   (C) 居民對於外面的世界非常嚮往，希望能出去
   (D) 是一個嚴刑峻法，秩序井然的法治社會
   解析：(B)。文中描寫「黃髮垂髫，並怡然自樂」，且不知漢魏，是避秦時亂而來，非神仙亦非嚮往外界。

2. (非選) 文末安排「太守遣人迷路」與「劉子驥規往未果」的情節，有何用意？
   參考答案：
   增添桃花源的神秘感與「理想難尋」的幻滅感，暗示這樣的理想世界在現實中是不存在的，或者是可遇而不可求的。
